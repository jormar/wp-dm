== Description ==

With Analytic Stats you can track your visitors in a simple and effective way.

*To see stats and analytics of your site login to WP Panel -> Stats*

7. Visitor Spy

== Installation ==

1. Download Analytic
1. Upload `analytic/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

After an update, please deactivate and activate the plugin to make sure all database settings are up to date.
